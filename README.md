# Prediksi-Saham-Perbankan-dan-Pertambangan

Proyek ini merupakan bagian dari penelitian saya sebagai Asisten Penelitian Mahasiswa (Undergraduate Research Assistant), yang berfokus pada pengembangan model deep learning untuk melakukan multi-step forecasting harga saham selama satu bulan ke depan dalam format full candlestick dengan menggunakan model LSTM.

Tujuan Penelitian
- Mengembangkan model deep learning untuk forecasting harga saham dalam format candlestick (Open, High, Low, Close).
- Melakukan multi-step prediction untuk horizon 22 hari ke depan, lebih kompleks dibanding prediksi next-day atau grafik garis biasa.

Library yang digunakan 
- Python (NumPy, Pandas, Matplotlib)
- TensorFlow / Keras
- Scikit-learn.  
- Menganalisis hasil evaluasi berdasarkan metrik error seperti **MAE**, **RMSE**, dan **MAPE**.
